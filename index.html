Apa Itu HTML?

HTML (Hypertext Markup Language) adalah bahasa markah standar yang digunakan untuk membuat dan merancang halaman web. HTML memberikan struktur dasar dan konten pada halaman web, yang kemudian dapat dilihat melalui peramban web seperti Chrome, Firefox, atau Edge.

Elemen Dasar HTML

HTML terdiri dari elemen-elemen yang dikelilingi oleh tag-tag. Setiap tag dimulai dengan tanda < dan diakhiri dengan tanda >. Contohnya, <p> digunakan untuk paragraf, <h1> untuk judul utama, dan <a> untuk tautan.

Struktur Dasar Halaman HTML

Setiap halaman HTML biasanya dimulai dengan elemen <!DOCTYPE html> yang mendefinisikan jenis dokumen. Kemudian, elemen <html> menandai awal dan akhir konten HTML. Di dalam <html>, terdapat <head> yang berisi informasi tentang halaman seperti judul dan tautan ke stylesheet, serta <body> yang berisi konten yang akan ditampilkan pada halaman web.

Penggunaan Elemen Lainnya

Selain elemen-elemen dasar, HTML juga memiliki berbagai elemen lain seperti <img> untuk menampilkan gambar, <ul> dan <ol> untuk membuat daftar tak berurutan dan berurutan, serta masih banyak lagi.

Belajar Lebih Lanjut

Jika Anda ingin mendalami HTML, Anda dapat mempelajari lebih lanjut tentang elemen-elemen lainnya, tata letak halaman dengan CSS, serta interaksi dinamis menggunakan JavaScript.
